# Getting Started

## Pre-Requisites
1. uv
2. npm

## Setup Google Gemini API Key
1. Open terminal 1
2. Change directory `cd adk-agent`
3. Create .env file from example `cp .env.example .env`
4. Edit `.env` and provide the Google Gemini API key.

## Starting ADK Agent
1. Open terminal 1
2. Change directory `cd adk-agent/adk_web_agent`
3. Execute `uv run main.py`

## Starting CopilotKit App
1. Open terminal 2
2. Change directory `cd copilotkit-app`
3. Execute `npm run dev`

## Launch Chabot in Browser
1. Open in chrome browser http://localhost:3000
